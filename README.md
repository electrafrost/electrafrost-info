# electrafrost.com

The open, machine-readable intellectual provenance of Electra Frost.

A verifiable record of ideas, contributions and predictions spanning 25 years (2000–2026) — from creative industries accounting to constitutional accountancy in the agentic economy.

## Stack

- React 18 + Vite
- Static JSON data
- Vercel deployment
- No external dependencies
